ember-leaflet-pm
==============================================================================

Use [leaflet.pm](https://github.com/codeofsumit/leaflet.pm) in Ember with [ember-leaflet](https://ember-leaflet.com) to create and edit geometry layers.


Installation
------------------------------------------------------------------------------

```
ember install ember-leaflet-pm
```


Usage
------------------------------------------------------------------------------

```hbs
<LeafletMap …>
  …
  <LeafletPmControl />
</LeafletMap>
```
